# 博客第三个版本

## 技术选型

### Docker 部署

> `tar --exclude=node_modules --exclude=dist --exclude=.nuxt --exclude=.git -zcvf ../blog3.tar.gz .`   
> `docker-compose up --build`  
> `docker-compose up -d`  


### 服务端 

> 使用 `nestjs` + `mongodb` + `redis` + `cos`

### 管理平台

> 使用 `Vuejs` + `Element UI`

### 博客前端

> 使用 `Nuxtjs` + `Tailwindcss`
